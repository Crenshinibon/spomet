Welcome to Spomet

Spomet is the contraction of Spotting Meteors.

It is a simple, but for many cases sufficient, search engine for Meteor. It should be easily includable into your Meteor project. See Spomet's GitHub repository for more information

Test It

Add text (taken from my blog) from the left side to the index, by using the provided button. Wait a moment, then you should be able to find the document, using the search box on the right.
